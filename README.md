# SkynetAccessibility Scanner

## Overview

SkynetAccessibility Scanner is a powerful ClassicPress accessibility testing and scanning plugin designed to help organizations identify, monitor, and fix accessibility issues across their websites. It enables businesses, agencies, and government organizations to maintain compliance with global accessibility standards while improving user experience for all visitors.

Built for ClassicPress plugin provides automated scanning, detailed reporting, and continuous monitoring - making accessibility management simple and scalable.

## Why use Automated ClassicPress accessibility monitoring plugin - SkynetAccessibility Scanner?

Accessibility is not a one-time task. With evolving standards and legal requirements, websites require continuous monitoring and improvements. This plugin helps you:

- Detect accessibility issues automatically
- Track compliance across multiple pages
- Prioritize fixes with actionable recommendations
- Maintain long-term accessibility compliance

It acts as both an accessibility checker and compliance testing tool, helping teams proactively manage accessibility risks.

## Installation

1. Load via installer or unzip the zip file in the `/wp-content/plugins/` directory, the folder must be called "skynetaccessibility-scanner".
2. Log into the ClassicPress admin panel.
3. Go to **Plugins > List**, and activate the SkynetAccessibility Scanner plugin.

## ⚙️ Configuration

After activation, go to **Plugins > SkynetAccessibility Scanner > Settings** to configure the SkynetAccessibility Scanner Report.

## CORS Policy Configuration

To avoid CORS policy issues, ensure the following URLs are allowed in your website. These URLs should be added to your CORS configuration or trusted domains list.

| **Domain**                            | **Description**                                    | **Usage**                        |
|---------------------------------------|----------------------------------------------------|----------------------------------|
| `https://skynetaccessibilityscan.com` | Skynet Accessibility Scan (Global Domain)          | API access and resources         |

## Instructions

1. Update your server's CORS configuration to include these URLs.
2. Ensure wildcard subdomains (`*`) are supported where necessary.
3. Verify the application functionality by testing requests to these domains.
4. If issues persist, consult the documentation for CORS configuration guidance.
